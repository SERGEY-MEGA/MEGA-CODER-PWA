# 🚀 MEGA CODER PWA

**Суверенная отечественная IDE — теперь в вашем браузере и на смартфоне.**

![MEGA CODER Banner](https://mega-coder.vercel.app/apple-touch-icon.png)

MEGA CODER — это интеллектуальная среда разработки нового поколения, спроектированная для полной автономности и высокой производительности. Теперь доступна как Progressive Web App (PWA), что позволяет вам программировать везде: от мощного десктопа до обычного смартфона.

---

## ⚡️ Быстрый старт

Откройте IDE в один клик и установите её на свое устройство:

[![Open in MEGA CODER](https://img.shields.io/badge/Открыть-MEGA--CODER-blue?style=for-the-badge&logo=vercel)](https://mega-coder.vercel.app)

## ✨ Основные возможности

* **PWA Ready**: Установите IDE на iPhone, Android или ПК как нативное приложение.
* **Offline First**: Работайте без интернета благодаря кэшированию и локальному хранилищу IndexedDB.
* **AI Pair Programmer**: Интеграция с передовыми моделями (Gemini 2.0 Flash, DeepSeek, YandexGPT).
* **Local LLM**: Поддержка WebLLM и Ollama для полностью приватной разработки на вашем устройстве.
* **Кроссплатформенность**: Адаптивный интерфейс, который подстраивается под любой экран.
* **Суверенность**: Полный контроль над вашим кодом и инструментами.

---

## 📱 Как использовать на смартфоне

1. Перейдите по ссылке [mega-coder.vercel.app](https://mega-coder.vercel.app).
2. Нажмите **«Поделиться»** (iOS) или **«Три точки»** (Android).
3. Выберите **«Добавить на главный экран»**.
4. Запускайте MEGA CODER как обычное приложение из меню телефона!
