# 📄 Документация MEGA CODER PWA

Добро пожаловать в официальную документацию MEGA CODER.

## 🏗 Архитектура приложения

MEGA CODER построен на базе **Next.js 14** и использует архитектуру **Modern PWA**.

### Основные компоненты системы

1. **Frontend Shell**: React-приложение с использованием Tailwind CSS для адаптивного дизайна.
2. **Virtual File System**: Реализована на базе **IndexedDB**, что обеспечивает персистентность данных без синхронизации с облаком.
3. **Cross-Platform API Wrapper**: Единый слой интеграции, который переключает методы взаимодействия (File System Access API vs Tauri API) в зависимости от среды запуска.
4. **Network Proxy Layer**: Позволяет выполнять запросы к внешним ИИ-провайдерам, обходя ограничения CORS в браузерной среде.

## 🔐 Безопасность и Приватность

Мы придерживаемся концепции **Суверенной IDE**:

* **Локальное хранение**: Ваш код никогда не покидает ваше устройство без вашего ведома.
* **Безопасный режим (Secure Mode)**: По умолчанию IDE работает полностью изолированно. Интернет-соединение требуется только для работы облачных ИИ по вашему требованию.
* **API Ключи**: Все ключи API (Gemini, DeepSeek и др.) сохраняются только в `localStorage` вашего браузера в зашифрованном виде.

## 🚀 Установка

### Требования

* Любой современный браузер (Chrome, Safari, Edge, Firefox).
* Поддержка Service Workers (стандарт для современных систем).

### Инструкция для установки на ПК

1. Откройте [https://mega-coder.vercel.app](https://mega-coder.vercel.app).
2. Нажмите иконку установки в адресной строке.

### Инструкция для мобильных устройств

* **iOS**: Safari → Поделиться → На экран «Домой».
* **Android**: Chrome → Меню → Установить приложение.
